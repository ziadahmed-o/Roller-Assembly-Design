# ⚙️ Heavy-Duty Roller Assembly Design

An industrial-grade, parametric 3D mechanical assembly of a heavy-duty roller conveyor unit. This project showcases precise component modeling, advanced assembly mating, and a detailed exploded-view configuration demonstrating the fastening stack-up using **SolidWorks**.

## ⚙️ Engineering & Design Specifications

* **Assembly Type:** Mechanical Roller / Conveyor Sub-Assembly.
* **Key Components:**
  * **Perforated Cylindrical Rollers:** Five high-traction textured steel rollers aligned for smooth rotational load distribution.
  * **Side Support Plates (Red):** Heavy-duty structural steel plates designed to house the roller shafts and secure the assembly structure.
  * **Structural Guide Rails (Yellow):** Precision-machined alignment tracks sitting beneath the main plate.
  * **Top Mounting Plate (Cyan):** Thick structural steel top plate acting as the main interface for load application.
  * **Standard Fasteners:** Fully integrated standard hexagonal flange bolts, flat washers, and countersunk screws configured directly from the SolidWorks Toolbox to ensure realistic thread mating and clearance.

---

## 🛠️ SolidWorks Features & Skills Demonstrated

* **Parametric Part Modeling:** Designing customized components including the perforated roller pattern and slotted mounting rails.
* **Advanced Mates:** Utilizing concentric, coincident, and distance mates to align multi-axial rollers and side plates with zero interference.
* **Exploded View Configuration:** Developed a logical step-by-step assembly/disassembly sequence detailing how the fasteners secure the top plate, guide rails, side plates, and roller shafts.
* **Toolbox Integration:** Utilizing standard ISO/DIN mechanical fasteners (Bolts, Washers, and Screws) with correct sizing to match mechanical design constraints.
